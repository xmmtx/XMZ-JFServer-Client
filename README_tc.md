## 簡介
本專案是Minecraft季落秋風伺服器官方用戶端，也是XMZ系列整合包之一。

## 使用方法
首先，執行Minecraft: Java Edition需要安裝Java。該包使用 `mc1.21.8`，建議直接安裝最新Java發行版。<br>

**Java**下載：<br>
[甲骨文 JDK25 Windows](https://www.oracle.com/cn/java/technologies/downloads/#jdk25-windows)<br>
[Azul JDK25 Windows](https://www.azul.com/downloads/?version=java-25&os=windows&package=jdk#zulu)<br>

> [!TIP]
> 以下命名方法僅適用於 `v3.0.0` 及以上版本

#### 使用自動更新包（特別推薦）
1. 前往發行版下載，檔案後綴為 `Auto`
2. 在合適的路徑下解壓縮
3. 按兩下 `HMCL-version.exe` 檔案會自動安裝整合包
4. 登入遊戲帳號
5. 啟動遊戲
#### 使用普通包（推薦）
1. 前往發行版下載，檔案後綴為 `Regular`
2. 在合適的路徑下解壓縮
3. 按兩下 `HMCL-version.exe` 檔案會自動安裝整合包
4. 登入遊戲帳號
5. 啟動遊戲
#### 使用匯入包
1. 前往發行版下載，檔案後綴為 `Import`
2. 開啟Minecraft啟動器，找到安裝整合包
3. 選擇剛剛下載的檔案
4. 點擊安裝整合包
5. 登入遊戲帳號
6. 啟動遊戲
#### 使用完整包
1. 前往發行版下載，檔案後綴為 `Full`
2. 在合適的路徑下解壓縮
3. 按兩下 `HMCL-version.exe` 檔案
4. 登入遊戲帳號
5. 啟動遊戲
#### 手動安裝（不推薦）
1. 前往發行版下載，下載原始碼（也可直接複製 `main`）
2. 手動根據整合包版本安裝Minecraft、Fabric
3. 登入遊戲帳號
4. 啟動遊戲
