## 概要
このプロジェクトはMinecraft JFServerの公式クライアントであり、XMZシリーズのModpackの一つでもあります。

## Mod一覧
完全なMod一覧：https://github.com/xmmtx/XMZ-JFServer-Client/tree/main/.minecraft/mods
#### 主要Mod
1. **3Dスキンレイヤー** - 作者: tr7zw - [MCMOD](https://www.mcmod.cn/class/4618.html) - ライセンス: tr7zw
2. **Roughly Enough Items** - 作者: shedaniel - [MCMOD](https://www.mcmod.cn/class/1674.html) - ライセンス: MIT
3. ...

> このModpackにはMinecraft本体、Fabric、Javaは含まれていません。

## 使用方法
まず、Minecraft: Java Editionを実行するにはJavaのインストールが必要です。このパックは `mc1.21.8` を使用しているため、最新のJavaリリースを直接インストールすることをお勧めします。<br>

**Java** ダウンロード：<br>
[Oracle JDK25 Windows](https://www.oracle.com/cn/java/technologies/downloads/#jdk25-windows)<br>
[Azul JDK25 Windows](https://www.azul.com/downloads/?version=java-25&os=windows&package=jdk#zulu)<br>

> [!TIP]
> 以下の命名方法は `v3.0.0` 以降のバージョンにのみ適用されます。

#### 自動更新パックを使用する（最も推奨）
1. リリースページからダウンロードし、ファイル名の末尾が `Auto` のものを選択
2. 適切なパスに解凍
3. `HMCL-version.exe` ファイルをダブルクリックすると、Modpackが自動的にインストールされます
4. ゲームアカウントにログイン
5. ゲームを起動
#### 通常パックを使用する（推奨）
1. リリースページからダウンロードし、ファイル名の末尾が `Regular` のものを選択
2. 適切なパスに解凍
3. `HMCL-version.exe` ファイルをダブルクリックすると、Modpackが自動的にインストールされます
4. ゲームアカウントにログイン
5. ゲームを起動
#### インポートパックを使用する
1. リリースページからダウンロードし、ファイル名の末尾が `Import` のものを選択
2. Minecraftランチャーを開き、Modpackのインストールを見つける
3. ダウンロードしたばかりのファイルを選択
4. Modpackのインストールをクリック
5. ゲームアカウントにログイン
6. ゲームを起動
#### フルパックを使用する
1. リリースページからダウンロードし、ファイル名の末尾が `Full` のものを選択
2. 適切なパスに解凍
3. `HMCL-version.exe` ファイルをダブルクリック
4. ゲームアカウントにログイン
5. ゲームを起動
#### 手動インストール（非推奨）
1. リリースページからソースコードをダウンロード（`main` ブランチを直接クローンすることも可能）
2. Modpackのバージョンに基づいてMinecraftとFabricを手動でインストール
3. ゲームアカウントにログイン
4. ゲームを起動
