## Introduction
Ce projet est le client officiel du serveur Minecraft JFServer et fait également partie de la série de modpacks XMZ.

## Liste des Mods
Liste complète des mods : https://github.com/xmmtx/XMZ-JFServer-Client/tree/main/.minecraft/mods
#### Mods Principaux
1. **Skin Layers 3D** - Auteur : tr7zw - [MCMOD](https://www.mcmod.cn/class/4618.html) - Licence : tr7zw
2. **Roughly Enough Items** - Auteur : shedaniel - [MCMOD](https://www.mcmod.cn/class/1674.html) - Licence : MIT
3. ...

> Ce modpack n'inclut pas Minecraft lui-même, ni Fabric, ni Java.

## Utilisation
Tout d'abord, exécuter Minecraft: Java Edition nécessite l'installation de Java. Ce modpack utilise `mc1.21.8`, il est recommandé d'installer directement la dernière version de Java.<br>

**Java** Téléchargement :<br>
[Oracle JDK25 Windows](https://www.oracle.com/cn/java/technologies/downloads/#jdk25-windows)<br>
[Azul JDK25 Windows](https://www.azul.com/downloads/?version=java-25&os=windows&package=jdk#zulu)<br>

> [!TIP]
> La méthode de nommage suivante ne s'applique qu'aux versions `v3.0.0` et supérieures.

#### Utiliser le package de mise à jour automatique (FORTEMENT RECOMMANDÉ)
1. Aller dans les releases pour télécharger, le suffixe du fichier est `Auto`
2. Décompresser dans le chemin approprié
3. Double-cliquer sur le fichier `HMCL-version.exe` installera automatiquement le modpack
4. Se connecter au compte de jeu
5. Lancer le jeu
#### Utiliser le package standard (RECOMMANDÉ)
1. Aller dans les releases pour télécharger, le suffixe du fichier est `Regular`
2. Décompresser dans le chemin approprié
3. Double-cliquer sur le fichier `HMCL-version.exe` installera automatiquement le modpack
4. Se connecter au compte de jeu
5. Lancer le jeu
#### Utiliser le package d'importation
1. Aller dans les releases pour télécharger, le suffixe du fichier est `Import`
2. Ouvrir le lanceur Minecraft et trouver l'installation du modpack
3. Sélectionner le fichier qui vient d'être téléchargé
4. Cliquer pour installer le modpack
5. Se connecter au compte de jeu
6. Lancer le jeu
#### Utiliser le package complet
1. Aller dans les releases pour télécharger, le suffixe du fichier est `Full`
2. Décompresser dans le chemin approprié
3. Double-cliquer sur le fichier `HMCL-version.exe`
4. Se connecter au compte de jeu
5. Lancer le jeu
#### Installation manuelle (NON RECOMMANDÉ)
1. Aller dans les releases pour télécharger le code source (vous pouvez également cloner directement `main`)
2. Installer manuellement Minecraft et Fabric selon la version du modpack
3. Se connecter au compte de jeu
4. Lancer le jeu
